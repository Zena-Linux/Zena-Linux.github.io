# Contact and Support

This page provides information on how to get help, report issues, and connect with the Zena community.

---

## Getting Help

### Discord Community

Join our Discord server for real-time support and community discussions:

[![Discord Invite](https://img.shields.io/badge/Join_Discord-Chat_with_us-blue?style=social&logo=discord)](https://discord.gg/22MFgGgjG9)

**Discord Features:**
- General chat
- Support channels
- Announcements

### GitHub Issues

For bug reports and feature requests:

1. Visit the [GitHub Issues](https://github.com/zena-linux/zena/issues) page
2. Search for existing issues
3. Create a new issue with the appropriate template

#### Bug Report

When reporting bugs, include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- System information
- Relevant logs

#### Feature Request

When requesting features:
- Use case description
- Proposed solution
- Alternative solutions
- Additional context

### GitHub Discussions

For questions and general discussion:

- Visit [GitHub Discussions](https://github.com/zena-linux/zena/discussions)
- Ask questions
- Share ideas
- Connect with community

## Security Issues

### Reporting Security Vulnerabilities

For security issues:

1. **Do NOT** open a public issue
2. Email maintainers directly
3. Or use GitHub's security advisories

**What to include:**
- Description of vulnerability
- Potential impact
- Proof of concept (if possible)
- Suggested fix (optional)
